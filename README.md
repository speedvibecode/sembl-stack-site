# sembl-stack-site

The website for [sembl-stack](https://github.com/speedvibecode/sembl-stack) — the
swappable, spec-driven software factory built around the
[sembl](https://github.com/speedvibecode/sembl) gate.

Static HTML/CSS/JS, no build step, deployed on Vercel. Sibling of
[sembl-site](https://sembl.vercel.app) and shares its industrial design system
(onyx surfaces, 1px beams, mono kickers) with a cyan-led identity.

The hero "demo" is a scripted replay of a real `sembl-stack loop` run — including
a first-iteration BLOCK and retry — implemented in `main.js` (no dependencies).

Local preview: any static file server, e.g. `python -m http.server`.
